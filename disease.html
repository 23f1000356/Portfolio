Disease Outbreak Prediction System

Project Overview

A Streamlit-based web app using machine learning to predict risks of Diabetes, Heart Disease, and Parkinson's Disease based on user inputs.

Core Features

  • Multi-Disease Prediction: Predicts three health conditions.
  • Interactive Interface: User-friendly Streamlit GUI.
  • Real-time Results: Instant predictions.
  • Input Validation: Handles invalid inputs.
  • Responsive Design: Works on all devices.

Supported Disease Predictions

  • Diabetes Prediction: Based on pregnancies, glucose, blood pressure, skin thickness, insulin, BMI, pedigree, age.
  • Heart Disease Prediction: Based on age, gender, chest pain, blood pressure, cholesterol, ECG, heart rate, angina, ST depression, vessels, thalassemia.
  • Parkinson's Prediction: Based on voice analysis features.
